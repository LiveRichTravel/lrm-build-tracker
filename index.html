How it works

Done inside the Claude desktop app (Cowork) · ~10–20 min
1

Install the skill

On today's build, tap Get the skill. It downloads a .skill file — open it / add it in your Claude app once and it's installed.

2

Run it in Claude

Make sure the OnlyFans connector is on, then just tell Claude to run it for one of your models. The skill already knows the steps.

3

Read the result

Claude pulls your numbers and builds the report in our LRM style. Read it over — that's the work the robot just did for you.

4

Prove it here

Copy what Claude gave you, paste it below as proof, add a screenshot, and hit Check it off. Done till tomorrow.
