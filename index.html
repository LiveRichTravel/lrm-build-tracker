How it works

Inside the Claude desktop app (Cowork) · ~10–20 min
1

Install the skill

On a build card, tap Get the skill. It downloads a .skill file — add it in your Claude app once and it’s installed.

2

Run it in Claude

Turn on the OnlyFans connector, then tell Claude to run it for one of your models. The skill knows the steps.

3

Read the result

Claude pulls your numbers and builds the report in our LRM style. That’s the work the robot just did for you.

4

Prove it here

Open the build, paste what Claude gave you as proof, add a screenshot, and hit Check it off.
