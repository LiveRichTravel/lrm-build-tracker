Set up a Chatter Grader yourself

No tech skills needed, about 5 minutes per model. This is just another job you hand to your Cowork: it reads the live numbers every couple of hours and posts a short coaching scoreboard into the model's team channel, so the chatters can adjust mid-shift.

🎯What it does

Every 2 hours during the shift it reads the model's live OnlyFans numbers and posts a short coaching scoreboard into her team's Slack channel: what's working, what to tighten, which whales to chase, and how today's money compares to her average. It's a coach, not a callout.

It reads captions and numbers only. It never looks at the photos or videos, so it never overpromises.

🔌Before you start (one time)

In the Cowork app, make sure two things are connected (green): OnlyFans API and Slack. If they're connected, you're ready.

Every model's account ID and team channel lives in the private team roster your manager keeps. You don't type IDs from memory, you look them up there.

✅The 5 steps

  1. Open Cowork (the Claude desktop app).
  2. Look up the model in the private roster, copy her account ID and her team channel.
  3. Type the setup request (template below), filling in the blanks.
  4. Test it once first. Say: "Before scheduling, run it once now so I can see it." Read the post. Happy with the tone? Move on.
  5. Approve the popup. When you ask to schedule it, a box pops up to approve the recurring task. Click Approve. It now runs on its own.

📋Copy-paste template

Fill the amber blanks from the roster, then paste into Cowork.

Set up a Chatter Grader for [MODEL NAME].
  • OnlyFans account: [acct_… from the roster]
  • Post live to Slack channel: [#channel-name / channel ID from the roster]
  • Schedule: every 2 hours, 8am–10pm.
  • Grade: caption honesty (text only, no image claims), reply speed, whale coverage, and today's money vs her 30-day daily average, with a motivational push if she's behind.
  • Banned words: [her list].
  • Tone: supportive coach. Never name or shame an individual chatter. Post directly and send me the link.

🔧Changing or stopping one

  • Change timing: "Make Shantal's grader run every 4 hours instead," or "…only 10am–10pm."
  • Pause it: "Turn off the Shantal Chatter Grader."
  • See what's running: "List my scheduled tasks."

💡Good to know

  • The approval popup is normal. Anything scheduled asks you to approve once, just click Approve.
  • It only reads captions and numbers, never the images, so it can't and won't claim the video matches or doesn't match. It checks caption wording against the banned-words and no-false-advertising rules.
  • Real numbers only. If a data pull fails, the post says so instead of guessing.
  • Scheduled tasks run while the Cowork app is open. If the computer is asleep when one is due, it runs at next open. For 24/7 coverage, keep it on an always-on machine.
Safety: never paste the OnlyFans API key or any password into the chat. It already lives inside the connector. The roster holds IDs only, and stays private to the team.
